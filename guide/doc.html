# 开发指南-文档部分

# 进入项目./docs，按如下格式新增

目录结构如下

    .
    ├─ components     // 为对应的组件文档
    │  └─ button.md  // 为对应的文档
    ├─ .vuepress
    │  └─ components  // 为对应的组件实例
    │     └─ button  // 为对应的组件实例
    │        └─ button1.vue  // 可以为多个实例，以便文档展示多种情况
    └─ config.js      // 菜单栏配置sidebar-'/'-children

# 注释说明

参数 说明
name 图表名称
Author 维护人
type 组件类型
img 预览图
desc 描述
src 文档路径
