# 象形柱状图

常用的象形柱状图。

# 基础用法

<template>
  <div style="width: 100%;height: 300px">
    <xw-pictorialBar
      :name="name"
      :seriesData="seriesData"
    ></xw-pictorialBar>
  </div>
</template>
<script>
  /*
  * name: 象形柱状图
  * Author: ha
  * type: bar
  * img: /img/pictorialBar1.png
  * desc: 坐标轴反向
  * src: /components/pictorialBar.html
  * */
import XwPictorialBar from '../../../../src/packages/echars/src/pictorialBar.vue'

export default {
  components: { XwPictorialBar },
  data () {
    return {
      name: 'PictorialBar',
      seriesData: [{name: '南京', value: 891}, {name: '上海', value: 1220}, {name: '广州', value: 660}]
    }
  }
}
</script>

# 属性

参数 类型 默认值 是否必填 说明 图表公用
inverse Boolean false 坐标轴是否反向 公用
symbol String 'circle' 图形类型 参考官方文档 (opens new window) 公用
bottomSymbol String symbol 底图类型 参考官方文档 (opens new window) 公用
symbolSize Number 30 图形大小 公用
