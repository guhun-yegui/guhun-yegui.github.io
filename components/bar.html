# 属性

参数 类型 默认值 是否必填 说明 图表公用
width String 100% 图表宽度 公用
height String 100% 图表高度 公用
xData Array [] X轴数据 公用
yData Array [] Y轴数据 公用
unit String 100% 单位 公用
yLengend Array [] 多维数据的文案 公用
name String AreaLine true 图表类型 公用
options Object options 参考官方文档 (opens new window) 公用
